# Figma

This storybook instance uses figma/figspec for shared components, which can be toggled in the storybook toolbar's design tab.

`STORYBOOK_FIGMA_ACCESS_TOKEN` environment variable is required for this to work.

To view the figma designs and prototypes, follow this <a href="https://www.figma.com/file/Zqo3TlKsRbPotFFkBlRq6g/Dashboard2.0?type=design&node-id=0-1&mode=design" target="_blank">Link</a>
